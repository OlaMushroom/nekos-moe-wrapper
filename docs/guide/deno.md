# For Deno Users

### By adding package...

::: code-group
```sh [NPM]
deno add npm:nekos-moe
```
```sh [JSR]
deno add @om/nekos-moe
```
:::

### ...or using import specifier in code
::: code-group
```ts [NPM]
import * as nekos from 'npm:nekos-moe';
```
```ts [JSR]
import * as nekos from 'jsr:@om/nekos-moe';
```
:::
